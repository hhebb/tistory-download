난이도

프로그래머스 level 2

유형

단순 구현
정렬
문자열 다루기

특이사항

카카오 문제답게 문자열 다루는 것이 까다롭다.
아이템이 하나일 때 예외 처리도 필요
인덱스 정렬에 딕셔너리를 활용함.

총평

전처리 부분
로직 부분
2 부분으로 나눠진다.
전처리를 잘 해놔야 로직에 집중할 수 있다.
문자열을 잘만 처리하면 어렵지 않음.

def solution(s):
    answer = []
    info = {}

    container = [i for i in s.split('},')]

    # 하나만 있으면 바로 반환
    if len(container) == 1:
        #info[1] = s[2:-2]
        answer = int(s[2:-2])
        return [answer]

    # s 받아와서 처리해주기    
    for idx, con in enumerate(container):
        if idx == 0:
            splited = con[2:].split(',')
            info[len(splited)] = splited
            #container[idx] = con[2:]
        elif idx == len(container) - 1:
            splited = con[1:-2].split(',')
            info[len(splited)] = splited
            #container[idx] = con[1:-2]
        else:
            splited = con[1:].split(',')
            info[len(splited)] = splited
            #container[idx] = con[1:]

    info = sorted(info.items(), key=lambda x:x[0])

    for i in info:
        for j in i[1]:
            check_num = int(j)
            if check_num not in answer:
                answer.append(check_num)
                break
            else:
                pass
    return answer
