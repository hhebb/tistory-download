점수가 두자리인 10 일 때 처리 까다로움

점수 슬롯을 만들어 점수계산 관리

총평

약 30 분 가량 걸림.
로직을 구현하는 구현 문제.
쉬운문제인데 문자열 다루기가 까다롭다고 생각함.
다른 사람 코드를 보고 배워야 함.

토큰화 하는 방법도 있는데, 길이가 짧아서 그럴 필요는 없는 듯. 

만약 한다면 정규표현식 re 를 써야함.

def solution(dartResult):
    answer = 0
    sub_result = [0] * 3
    order = -1

    for idx, c in enumerate(dartResult):
        if ord(c) < 58 and ord(c) >= 48:
            # 10 이 었을 때 넘겨야 함.
            if idx > 0 and (ord(dartResult[idx-1]) >= 48 and ord(dartResult[idx-1]) < 58):
                continue

            order += 1

            # 10 인 경우
            if idx < len(dartResult) and dartResult[idx+1] == '0':
                sub_result[order] = 10
                print(10, end='')
            # 10 미만 점수인 경우
            else:
                sub_result[order] = int(c)


        elif ord(c) > 65 and ord(c) < 97:
            if c == 'D':
                sub_result[order] **= 2
            elif c == 'T':
                sub_result[order] **= 3
        else:
            if c == '*':
                sub_result[order] *= 2
                if order > 0:
                    sub_result[order - 1] *= 2
            else:
                sub_result[order] *= -1
    #print(sub_result)
    answer = sum(sub_result)

    return answer
