SSH

개념

ssh 는 secure shell 의 약자이다.
네트워크를 이용해 다른 device 에 접속하려고 할 때 사용되는 프로토콜 혹은 그것이 구동되는 프로그램 정도로 이해하면 된다.
과거의 telnet 등의 접속 프로토콜을 대체한다.

특징

RSA 를 이옹한 암호화로 인해 아직까지 매우 보안성이 뛰어나다고 한다.

방식

접속을 요청하는 컴퓨터에 ssh key 를 만들면 공개키, 개인키 1 쌍이 만들어진다.
이 key 들은 수학적인 관계가 있는데, 거의 유일한 쌍이다.
개인키는 가지고 있고, 접속하려는 컴퓨터에 만들어진 공개키를 복사해서 등록해준다.
그러면 접속할 때마다 이 개인 키를 보내서 비교를 하여 나인지 아닌지 판별한다.

기타

리눅스, 윈도우 모두 ssh 기능이 있고 key 를 만드는 작업을 터미널 등에서 할 수 있는 듯하다.
