출처

프로그래머스

난이도

프로그래머스 level 2

유형

완전 탐색
DFS, BFS

특이사항

매 번 둘 중 하나를 선택하는 전형적인 탐색 문제.

총평

완전 탐색으로 시간초과 날 줄 알았는데 통과해서 그대로 둠.
익숙한 문제는 재귀를 쓸 줄 알지만 아닌 것은 아직 어려움.
어떤 정보를 인자로 넘겨줘야할지 판단이 안 된다.

def solution(numbers, target):
    answer = 0
    if len(numbers) == 0:
        if target == 0:
            return 1
        else:
            return 0

    next_num = numbers.pop(0)
    answer += solution(numbers, target - next_num)
    answer += solution(numbers, target + next_num)
    numbers.insert(0, next_num)
    return answer
