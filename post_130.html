GCD, LCM

최대공약수와 최소공배수

GCD (Greatest Common Divide)

  • 개념
    두 수의 최대공약수
    유클리드 호제법 알고리즘을 그대로 구현한다.
  • 비고
    무식하게 숫자 하나씩 늘려가며 구하면 $O(N)$ (N = 두 수 중 작은 수) 의 시간 복잡돌이지만
    유클리드 호제법을 이용하면 $O(logN)$ 에 끝난다.
    사실 시간복잡도는 더 엄밀하게 따져야 정확하지만 대략 log 로 줄어든다.
# GCD 재귀 구현
def gcd(a, b):
    if b == 0:
        return a
    return gcd(b, a%b)

# GCD 반복문 구현
def gcd_2(a, b):
    while b > 0:
        a, b = b, a % b
    return a

# 3 개 이상 수의 GCD 구하는 함수
def gcd_multi(arr):
    res = arr[0]
    for i in arr:
        res = gcd(res, i)
    return res

LCM (Least Common Multiply)

  • 개념
    두 수의 최소공배수
    GCD 를 알고 있으며 즉시 구할 수 있다.
  • 비고
    $a * b = GCD(a, b) * LCM(a, b)$ 이므로 나누고 곱하기만 해주면 된다.
# LCM 구현
def lcm(a, b):
    return int(a * b / gcd(a, b))

# 3 개 이상 수의 LCM 구하는 함수
def lcm_multi(arr):
    res = 1
    for i in arr:
        res = lcm(res, i)
    return res
