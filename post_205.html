템플릿 엔진

  • 흔히 웹개발에서 HTML 를 많이 쓰는데, 이를 확장한 언어를 템플릿이라고 한다.
  • 순수 HTML 언어보다 템플릿을 이용하면 변수나 데이터를 넣기 편하고 동적으로 만들기 좋다.
  • 서버 사이드 언어에서 주로 쓴다.
  • 요즘엔 프론트엔드 프레임워크에서도 자주 쓰인다.
  • 순수 HTML 보다 더 많이 쓰이는 듯 하다.

종류

  • node.js - pug(jade), ejs
  • java - jsp, freemarker
  • react - jsx
