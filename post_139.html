문자열 매칭 2

Trie 구조

  • 개념
    찾으려는 문자열을 고속으로 탐색할 수 있는 일종의 트리 형태.
    트리의 노드에는 문자 하나가 들어가고,루트부터 리프 노드 까지는 하나의 문자열에 해당하고, 그 사이는 접두사에 해당함.
    키워드 트리라고도 한다

  • 비고
    최대로 찾으려는 문자열의 길이만큼 시간이 걸린다.
    시간이 감소되지만 문자열을 다 저장하는데 공간이 매우 많이 필요하게 된다.
    trie 는 기본적으로 문자열 삽입, 문자열 탐색, 접두사 탐색 연산 등을 지원한다.
    배열에 저장하고 비교하려면 점점 시간이 오래 걸리지만 트라이에 접두사 기준으로 추가해주면 시간을 아낄 수 있다는 점을 이용하는 것이다.

# trie
class Node:
    def __init__(self, data, total=None):
        self.data = data
        self.total = total
        self.child = {}


class Trie:
    def __init__(self, data):
        self.root = Node(data)

    def insert(self, s):
        cur_node = self.root
        for c in s:
            if c not in cur_node.child:
                cur_node.child[c] = Node(c)

            cur_node = cur_node.child[c]
        cur_node.total = s


    def search(self, s):
        cur_node = self.root
        find = True
        for c in s:
            # 현재 노드의 자식 중에 비교하려는 문자가 없을 때 찾는 문자열이 없다.
            if c not in cur_node.child:
                return False
            cur_node = cur_node.child[c]

        # 정확히 같은 길이인 찾으려는 문자열이 다 있을 때 True
        if cur_node.total:
            return True
        else:
            return False

    def search_with(self, prefix):
        cur_node = self.root
        # 트라이가 접두사를 포함하는 문자열을 저장하고 있는지 판단.
        for c in prefix:
            cur_node = cur_node.child[c]
            if cur_node.data != c:
                break

        # 접두사 포함 문자열이 있을 때, bfs 를 돌며 모든 문자열을 반환한다.
        q = [cur_node]
        res = []
        while q:
            here = q.pop(0)
            # here 이 마지막 문자일 때, 결과값에 넣어준다.
            if here.total:
                res.append(here.total)
            for k, node in here.child.items():
                if node.total:
                    # 자식 중 마지막 문자일 때, 결과값에 넣어준다.
                    res.append(node.total)
                else:
                    q.append(node)
        return res

trie = Trie('')
trie.insert('seoul')
trie.insert('seul')
trie.insert('set')
trie.search('seoul')
trie.search_with('se')
