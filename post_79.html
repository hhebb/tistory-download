프로젝트 구상

앞으로 웹 개발을 하면서 포트폴리오, 취미, 서비스 등을 위한 나만의 프로젝트들의 리스트업.
개발자를 오래할 것이라고 생각하고 장기적 관점에서 작성.
모든 것은 포트폴리오라고 할 수 있지만, 오로지 포트폴리오만 목적으로한 것은 따로 뺐다.
사실 웹 말고 그냥 코드 스니펫 형태로 만드는게 나은것도 있다.
근데 그냥 아이디어를 던져 봄.

서비스

  1. 파일 변환 및 변형 서비스 웹 사이트
  2. 이미지 픽셀화, 아스키 아트 변환 서비스 웹 사이트
  3. visualgo 같은 웹 기반 비주얼 알고리즘 학습 사이트
  4. 울프람 알파 같은 유틸 웹인데 특수한 연산을 시각화해주는 사이트
  5. 설문지, 테스트 같은 페이지
  6. 타자연습 페이지
  7. 명언 제조기 사이트 - 컨텐츠 생산의 성격임.
  8. 웹소켓으로 거래소 가격 모니터링
  9. 또는 tradingView api 로 차트 띄우기
  10. json, xml valid 검사
  11. 구글 익스텐션 만들기
  12. 크롤러, 스크래퍼

포트폴리오(포트폴리오 목적만)

  1. 코로나맵 같은 지도관련 서비스.
  2. 가상화폐 거래소별 가격 모니터링 사이트
  3. 가족들과 행사, 사진 등을 올리는 커뮤니티 사이트
  4. 친구들과 사진 업로드, 잡담, 여행 계획 등을 하는 커뮤니티 사이트
  5. 공부한 것 질문 형식으로 문제 띄우는 것 (이건 앱이 더 적합)
  6. 게시판
  7. 소개 페이지
