정적 사이트

  • DB 없이 html, css, js 만으로 이루어진 사이트.
  • 사실상 서버의 역할이 필요없다.
  • 개인 블로그나 소개 페이지, 포트폴리오에 적합하다.
  • 적당한 저장소만 있으면 되는 성격의 웹사이트다.

정적 사이트 생성기

  • 최근에는 이러한 정적 사이트를 이쁘게 만들어주는 플랫폼이 꽤 보인다.
  • 얼마전까지 루비로 작성하는 jekyll 이 인기가 많았는데 요즘은 gatsby 가 더 좋은것 같다.
  • 루비 대신 node 환경에서 작성하고 또 react 를 이용해 더 이쁜 페이지를 만들 수 있다.
  • 빌드와 배포도 netlify 나 heroku 와 궁합이 잘맞아 더욱 인기가 많다.

플랫폼

  • jekyll, gatsby, hugo, ghost 등이 있다.
  • ghost 는 서비스가 좋은데 완전 유료로 바뀜. (비싸진 않음)
  • gatsby 는 node 를 사용하고 react 기반이라 좋은데 테마가 별로 없음.
    테마를 내가 직접 만들지는 않을 것임
  • hugo 는 테마가 많아서 좋다. 이걸로 정한다.
