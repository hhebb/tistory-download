유형

단순 구현

특이사항

문제 설명이 조금 부실하다고 생각됨.
딕셔너리 반복문에서 key, value 뽑아냄.
최댓값은 단순히 max 내장 함수 씀.

총평

문제 이해를 못해서 힌트를 봤다.
문제 전체 수준은 낮은데 설명이 너무 모호했다.
딕셔너리 활용도는 점차 익숙해져간다.

def solution(citations):
    answer = 0
    sorted_cite = sorted(citations, reverse=True)
    dict_cite = {}
    h = []

    for i in range(len(sorted_cite)):
        dict_cite[sorted_cite[i]] = i + 1

    for k, v in dict_cite.items():
        h.append(min(k, v))
    answer = max(h)
    return answer
