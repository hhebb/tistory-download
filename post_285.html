ssl

  • Secure Socket Layer
  • 네트워크 전송을 암호화하는 기술
  • 인증, 암호화, 복호화 기술을 지원한다.
  • 브라우저 뿐 아니라 서버, 이메일의 보안도 담당할 수 있다.

https

  • 앱 층에서 ssl 구현
  • 사이트 서버에서 인증서를 발급받는것이 필요하다.
  • 참고로 직접 사려면 좀 비싸다.
  • 그냥 호스팅 업체에서 인증서 사는게 더 싼듯.

tls

  • 전송 계층에서 ssl 구현
  • https 보다 더 강력하다고 한다.
  • 하지만 그냥 SSL 이라고 뭉뚱그려 지칭한다.
