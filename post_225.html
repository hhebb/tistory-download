고통받는 자

  • 윈도우를 사용하는 사용자나 개발자들은 한글 이름 폴더 때문에 고통받은 적이 있었을 것이다.
  • 사용자 폴더는 시스템에서 엄청 중요하다.
  • 개발에 필요한 라이브러리 관리 등 여러가지 작업을 해주기도 한다.
  • 그런데 이게 한글이면 골치아픈 일이 많이 생긴다.

드디어 해결

  • 예전에는 그냥 작업폴더를 C 로 옮겨서 꾸역꾸역 진행했는데 이제 도저히 그 방법으로 통하지 않는 판국이 되었다.
  • 미루고 미루던 사용자 폴더 이름 변경을 했다.
  • 사실 예전에도 찾아봤었는데 쉽지 않은 작업이라고 들어서 포기했었다.
  • 윈도우 사용자 이름을 바꾼다고 되는것은 아니다.
  • 위의 블로그를 따라했지만 정확히 어떤 원리로 되는것인지는 모르겠다.
  • 어쨌든 성공적으로 사용자 폴더 이름을 영어로 변경했다.
  • 중요한건 환경변수도 다 바꿔줘야한다.
  • 당연히 사용자폴더가 포함된 환경변수가 있을 것이다.
  • 이것은 자동으로 바꿔주지 않으므로 일일이 바꿔줘야한다.
