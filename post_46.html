총평

처음에 조합문제 인것 같아서 combinations 썼지만 시간초과남.
입력값이 100,000 이라서 단순히 O(N) 으로 풀어야겠다고 생각함.
중복이 안되게 하는 간단한 해시 문제임을 캐치하고 풀었음.

def solution(nums):
    answer = 0
    pool = set()

    for num in nums:
        pool.add(num)
        if len(pool) >= int(len(nums) / 2):
            break
    answer = len(pool)
    return answer
