용어

일급 객체 (first class object)

  • 개념
    프로그래밍 언어에서 일급 객체라고 하는 것이 있다.
    일급 객체는 다른 객체들에 일반적인 연산이 가능한 것이라고 한다.
    보통 변수에 대입, 파라미터에 넘기기, 함수값 반환 등도 포함된다.
    파이썬, js 에서는 함수를 일급 객체로 다룬다.
    함수 자체가 객체다.

스코프

  • 개념
    변수의 접근 및 참조 수준을 정의하는 것이다.

  • 종류

    1. 함수 레벨 스코프
    • 함수 내에 선언된 변수만 지역 변수가 된다.
    • 나머지는 전역변수.
    1. 블럭 레벨 스코프
    • 함수 뿐 아니라 코드 블럭 안에서 선언된 변수는 모두 지역 변수.

  • 분류
    c 계열 언어나 java 등은 모두 블럭 레벨 스코프
    고전 js 는 함수 레벨 스코프다.

호이스팅

  • 변수나 함수 선언을 스코프에서 가장 선두에서 한 것처럼 해 준다.
  • js 말고는 들어본 적이 없는 특이한 특성이다.
  • js 는 엔진이 모든 함수(선언 방식만)와 변수 선언을 호이스팅한다.
