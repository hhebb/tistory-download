웹 용어

SPA

  • single page application
  • 말 그대로 단일 페이지로 구성된 웹 페이지이다
  • 웹 서비스보다 앱에 더 가깝다.
  • 장점
    새로고침이 일어나지 않아 쾌적하다.
    빠르다.
  • 단점
    검색엔진 최적화에 안좋다.
    최초 구동 시간이 길다.
  • 구현법
    AJAX 등으로

프레임워크

  • SPA 만드는 것을 도와주는 프레임워크
    • React
    • Vue
    • Angular

라우팅

  • CSR 을 한다고 라우팅이 불가능하다는 것은 아니다.
  • SPA 프레임워크는 자체적으로 라우팅을 지원하여 일반적인 동적웹처럼 페이지 이동이 가능하다.
  • 차이점은 여전히 SPA 이며, 서버에서는 필요한 데이터만 제공받아 여전히 CSR 이라는 점이다.

리액트

  • 현재 본인은 리액트를 공부중이다.
  • 리액트는 vue, angular 와 다르게 프레임워크가 아니라 라이브러리다.
  • 리액트는 라우팅 기능을 제공하지 않는다.
  • react-router 를 사용하면 라우팅을 구현할 수 있다.
