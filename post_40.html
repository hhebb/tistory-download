총평

간단한 문제이다.
일반적으로 손으로 계산하는 순서와 조금 다르게 풀었다.
반복문이 어떻게 돌아가는지 머릿속으로 시뮬레이션하는 훈련이 필요.
변수와 인덱스가 많아지면 디버깅 소요시간이 기하급수적으로 늘어남.
지면에 반복문 수도코드와 문제 도식화한 것을 매칭 시켜놓는게 좋음.

def solution(arr1, arr2):
    answer = [[0] * len(arr2[0]) for i in range(len(arr1))]

    for idx, r1 in enumerate(arr1):
        for ii, elem in enumerate(r1):
            for idx2, r2 in enumerate(arr2[ii]):
                answer[idx][idx2] += elem * r2
    return answer
