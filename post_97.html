기본 정렬 유형.
기본이지만 반드시 짚고 넘어가야 하는 부분이다.
안정 정렬이라는 중요한 개념을 제대로 알고 있어야 한다.
버블 정렬, 삽입 정렬, 병합 정렬은 일단 자체가 안정 정렬이 된다.
파이썬에서 sort(key=lambda x:x[0]) 에 키를 설정하면 안정정렬이 된다는 것 같다.

나이순 정렬

  • 난이도 - 하
  • 유형 - 정렬
  • 핵심
    안정정렬이 핵심이다.
    단순 숫자가 아니라 2 개 이상의 정보를 정렬할 때 튜플을 사용한다.
    정렬 key 순서대로 튜플에 담아둔다.
  • 비고
    sorted 에 key 값을 설정하면 안정정렬이 적용된다.???
# 나이 순 정렬
n = int(input())
people = []
for i in range(n):
    tmp = list(input().split())
    print(tmp)
    people.append((int(tmp[0]), tmp[1]))

people = sorted(people, key=lambda x:x[0])

for i in people:
    print(i[0], i[1])

좌표 정렬하기

  • 난이도 - 하
  • 유형 - 정렬
  • 핵심
    첫 원소, 둘 쨰 원소 순서로 정렬해야하는데,
    파이썬 내장 라이브러리는 자동으로 순차적으로 정렬해준다. 
# 좌표 정렬하기
n = int(input())
coord = []
for _ in range(n):
    coord.append(list(map(int, input().split())))
coord.sort()
for c in coord:
    print(c[0], c[1])

수 정렬하기 3

  • 난이도 - 하

  • 유형 - 정렬

  • 핵심
    10,000,000 개면 nlogn 으로 못 푼다.
    계수정렬 O(n) 으로 풀어야 한다.
    단, 원소의 값이 어느정도 작을 때 적용할 수 있다.

  • 비고
    계수 정렬은 매우 단순하다.
    그냥 원소 최대값만큼 리스트를 만들고 거기에 등장하는 원소의 수를 카운트한다.
    그래서 원소 최대값이 적당히 작아야 한다. 100,000 대 면 쓸 만하다.
    원소를 한 번씩만 보면 되므로 O(N) 보장.
    계수 정렬의 아이디어(값을 인덱스에 매핑) 는 자주 등장하므로 중요하다!

# 수 정렬하기 3
import sys

n = int(sys.stdin.readline())#int(input())
arr = [0] * 10001
for _ in range(n):
    arr[int(sys.stdin.readline())] += 1
for idx, count in enumerate(arr):
    for j in range(count):
        print(idx)
