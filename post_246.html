자주 쓰는 명령어

ls

해당 디렉토리의 파일 목록 나열
-l 은 권한 등 자세한 정보. 일반 파일은 -, 디렉토리는 d 를 붙임.
-a 는 숨김 파일도 표시. '.' 이 붙으면 숨김파일이라는 것.

cd

디렉터리 이동.
~[user] 는 특정 사용자의 홈 디렉토리를 표현!

whoami

현재 유저 이름.

pwd

현재 디렉토리.
/ 는 루트 디렉토리

touch

빈 파일을 만드는 것
만약 기존의 파일을 touch 명령으로 만들면 생성시간만 바뀐다.
당연히 다른 디렉에 생성 가능

rm

파일 삭제
-i 옵션은 삭제 경고 띄움
-f 는 경고없이 삭제. 기본값.
다른 디렉토리에서 삭제 가능
여러개 삭제 가능

cp

파일 복사
다른 디렉토리로 복사 가능
다른 디렉토리의 파일을 또 다른곳으로 복사 가능

현재 디렉 표현

. 이나 ./ 로 적는다.

mv

파일 이동
cp 와 비슷한데 복사하지 않고 이동만 한다.
파일 이름만 바꾸는 데에도 사용할 수 있다.

mkdir

디렉토리 생성
-p 로 depth 있는 디렉토리 생성 가능.

rmdir

디렉토리 삭제.
비어있지 않으면 즉시 삭제 불가.

rm -r

즉시 디렉토리 삭제.
위험하다.

cat

파일 내용을 터미널에 출력

head, tail

파일의 맨 앞, 맨 뒤 출력
-[num] 하면 num 줄 만큼 출력

more

파일을 페이지 단위로 출력.
space 바 누르면 다음 페이지, enter 누르면 한 줄 씩

less

more 와 비슷하지만 더 편함.

file

어떤 파일 형식인지 출력
리눅스는 파일 확장자가 없는 경우가 많다.
그래서 확인하는것이 필요.

파일 이전

  • ssh 를 통해서 파일을 전송한다.
  • scp [src] [dst] - src host 의 path에서 dst host 의 path 로 파일을 전송.
  • -r - 파일이 아닌 폴더를 전송한다.
