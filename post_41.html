총평

예제 수준의 동적계획법 문제로 유명한 피보나치 문제.
단, 함수 스택 오버되므로 재귀로 풀면 안됨.
반복문으로 bottom up 형식으로 풀어야 함.
조금만 어려워지면 아직 풀기가 힘듬.
쉬운 문제도 DP, 재귀는 조금 부담스러움

def solution(n):
    answer = 0
    memo = [-1] * 100001
    memo[0] = 0
    memo[1] = 1

    for i in range(2, n + 1):
        memo[i] = memo[i - 1] + memo[i - 2]
    answer = memo[n]
    return answer % 1234567
