총평

문제 자체는 단순함.
하지만 카카오 문제 답게 대신 기타 알고리즘에 대한 지식이 있거나 예외 처리에 주의해야 한다.
카카오 문제는 마냥 쉬운 문제는 없다.
실력이 올라가더라도 어느정도 긴장을 하고 풀어야 함.

def solution(cacheSize, cities):
    answer = 0
    cache = []

    if cacheSize <= 0:
        answer = 5 * len(cities)
        return answer

    # 문자 전처리
    cities = [city.upper() for city in cities]

    for idx, city in enumerate(cities):
        # hit
        if city in cache:
            answer += 1

            cache_index = cache.index(city)
            # key 가 최신이면 패스
            if cache_index == len(cache) - 1:
                pass
            # key 가 오래된 것이라면 최신화. cache key 순서 바꿔주기
            else:
                for i in range(cache_index, len(cache) - 1):
                    cache[i] = cache[i + 1]

                cache[len(cache) - 1] = city
        # miss
        else:
            if len(cache) < cacheSize:
                pass
            else:
                cache.pop(0)
            cache.append(city)
            answer += 5

    return answer
