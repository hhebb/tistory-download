간단한 문제인데 처음에 오답 떠서 방황.
제대로 구현했는데 마지막 2 개 케이스 시간초과 떠서 당황
힌트보고 다시 구현함.
리스트 말고 변수 3 개 돌려막기로 해결.

cache = [-1] * 3
cache[0] = 1
cache[1] = 2
cache[2] = 0

def solution(n):
    for i in range(2, n):
        cache[2] = cache[1] + cache[0]
        cache[0] = cache[1]
        cache[1] = cache[2]

    answer = cache[2]%1000000007
    return answer
