IP 주소 알기

옛날부터 헷갈려서 머리가 아팠는데 이제서야 명확하게 알게 된다.

라우터

  • 패킷을 도착지로 보내주는 장비
  • 공인 IP 를 가진다.

모뎀

  • 최근에는 라우터와 일체형이다.
  • 주로 통신사에서 관리하고 공인 IP 를 가지게 된다.

공유기

  • 모뎀에 연결하여 무선 와이파이를 뿌려주는 장비
  • 무선 신호를 뿌려주는 것만 한다.
  • 라우터의 NAT 기능을 극대화한 것임.
  • 사설 ip 만 가지고 있다.
  • 공유기 자체도 사설 ip 를 가지는데 보통 1 번이다.

허브

  • 공유기에 많은 장치가 연결되면 속도가 느려질 수 있는데, 신호를 증폭해주는 장비.
  • 즉 공유기 설치 이후에 하는 것임.

기타

  • 진짜 ip 는 모뎀(라우터) 의 공인 ip 이다.
  • 만약 공유기에 와이파이로 무선 접속했다면 사설 ip 를 가진다.
    그 상태에서 ipconfig, ifconfig 하면 사설 ip 만 나온다.
    게이트웨이가 보통 공유기.
