기초 정렬 참고

몇 차례 공부를 했었지만 결국 내용을 잊어버리게 된다.
의미를 기억하고 코드를 외워보기도 했지만 잊어버린다.
알고리즘이란건 시간의 흐름에 따라 절차적으로 진행되는 것이라서 그 과정자체를 기억해야 한다.
정지된 문자가 아닌 최소한의 알고리즘이 동작하는 과정.

바로 생각나지 않으면 그냥 여기에 들어가서 동작 과정을 보면 된다.

 

버블 정렬

  • 개념 - 끝 점(i) 를 찍고 처음부터(j) 그 다음(j+1) 점과 비교하여 스왑하며 끝 점까지 진행.
  • 비고 - 안정 정렬
def bubble_sort(arr):
    for i in range(len(arr)-1, -1, -1):    # 끝점을 찍는다.
        swap = False
        for j in range(i):                 # 첫 지점부터 시작하여 다음 지점과 비교한다.
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swap = True                # 매 스텝마다 스왑 여부를 확인하여 조기종료를 할 수 있다.
        if not swap:
            break
    return arr

선택 정렬

  • 개념 - 첫 지점(i) 을 찍고 그 지점부터 끝까지(j) 가며 최소값을 찾아서 첫 지점과 바꾼다.
  • 비고 - 불안정 정렬
def selection_sort(arr):
    for i in range(len(arr)):                # 최소값을 놓은 첫점을 찍는다.
        min_idx = 0
        min_value = 1e9
        for j in range(i, len(arr)):        # 그 지점부터 이후까지 돌며 최소값을 찾는다.
            if arr[j] < min_value:
                min_value = arr[j]
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

삽입 정렬

  • 개념 - 두 번째 지점(i)부터 시작하여 그 앞에서 처음까지(j) 거꾸러 내려가면 보다 작은 값이 나올때까지 바꾼다.
  • 비고 - 안정 정렬
def insertion_sort(arr):
    for i in range(1, len(arr)):            # 두 번째 지점부터 찍는다.
        for j in range(i, 0, -1):            # 그 지점 앞에서 부터 첫 지점까지 거꾸로 자신이 들어갈 자리를 찾는다.
            if arr[j-1] > arr[j]:
                arr[j], arr[j-1] = arr[j-1], arr[j]
            else:
                break
    return arr
