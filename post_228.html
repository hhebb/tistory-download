무선 링크

wi-fi 성능

  • 성능은 AP 와의 거리에 좌우된다.
  • SNR
    신호 대비 잡음 비율.
    데이터를 담는 양인 coding rate 가 너무 높으면 에러에 취약하다.
  • BER
    비트 대비 에러 비율.
  • 네트워크 기능은 상당히 전력소모가 심해서 꺼두는게 좋다.

Cellular Network

  • 전지역을 cell 의 형태로 나눈다.
  • 첫 hop 만 무선이다.

셀룰러 네트워크 프로토콜

  • wi-fi 처럼 random access 가 아닌, 전통적인 채널 분할 방식을 사용한다.
  • CDMA - code division multiple access
    FDMA, TDMA 는 주파수나 시간을 분할한다.
    CDMA 는 다 섞어서 쏴버린다.
    수학적인 트릭, 코드를 이용해 유저가 해당 신호만 증폭시켜 들을 수 있도록 한다!
  • 3G 부터 사용하기 시작함.

모바일 네트워크

  • 계층적으로 구성되어서 실제 게이트웨이는 10 개도 안된다??
  • 셀 들이 스위치들처럼 계층적으로 구성되어 있다.
  • SGSN GGSN 등의 계층
  • 셀룰러 네트워크 만의 고유의 프로토콜을 사용한다.
