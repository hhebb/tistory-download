용어

차이점

  • URI
    Unified Resource Idenfier
    접근 프로토콜 + 도메인 및 주소(IP) + 자원(파일)의 이름
    자원의 고유한 식별자. 혹은 자원을 나타내는 규약의 개념이다.
    어떠한 파일이나 페이지의 주소 뿐 아니라 그 곳의 특정한 식별자까지 포함한다.
    URL 과 URN 은 URI 를 표현하는 방법에 해당한다.
  • URL
    Unified Resource Locator
    어떠한 파일이나 페이지의 위치까지만 나타낸다.
  • URN
    Unified Resource Name
    파일의 위치가 아닌 고유한 이름으로 나타낸다.
    파일이 여기저기 옮겨 다닐 때 URL 매치가 안되는 경우를 방지하기 위해 대두됨.
