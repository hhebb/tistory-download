css 기초

css 를 다루기 전에 필요한 기초 단위들

크기 단위

px - 픽셀. 고정 단위.
% - 퍼센트. 상대적인 단위. 비율에 해당하므로 가변적 레이아웃에 자주 사용됨.
em - % 와 같이 상대적인 단위. 글자 크기 기준. 동적인 페이지, 글자 크기에 따라 다른 크기가 필요할 때 사용.

색상 단위

색상이름 - 약 140 개 정도 이름으로 선택할 수 있음. 테스트용으로 사용함.
RGB - rgb(r,g,b)
hex - 16 진수 색상코드. #ff2830 등으로 작성

스타일을 적용하는 방법

link 요소로 연결 - html 과 css 를 분리하는 가장 권장되는 방법.
<link href=".css" rel="" type=".css"/>
style 요소로 연결 - css 를 분리하지 않고 html 내부에 css 를 작성. 테스트 용도로 씀.

선택자

스타일을 주기 위해서는 선택자를 통해 해당 요소를 선택한 뒤에 해당하는 스타일을 준다.

기본적인 선택자

기본적인 선택자들

요소 선택자

h1, img, p, div 등 요소 명을 적어서 나타내는 선택자.
페이지에 있는 전체 해당 요소에 영향을 준다.
p{color: red;} - p 요소의 색상을 건드린다.

id 선택자

id 속성을 가진 요소를 선택하는 선택자
이름앞에 # 을 붙여 표현.
_id{color: red;}

## class 선택자

class 속성을 가진 요소를 선택하는 선택자
이름앞에 . 을 붙여 표현
._class{color: red;}

선택자 중첩 사용

요소 선택자, id 선택자, class 선택자는 중첩 사용 가능하다.
p.box{} - box class 중 p 요소에만 영향을 줌
p#box{} - box id 중 p 요소에만 영향을 줌

선택자 묶음

h1, div, p{} - 선택자를 묶어서 요소들에게 영향을 줄 수 있다.
class 선택자, id 선택자 모두 묶기가 가능하다.

후손 선택자

중첩된 요소에 접근하기 위한 선택자
띄어쓰기로 구분한다.
.box2 .name{} - box2 class 의 name class 를 가지는 요소들을 선택한다.
중첩 선택자와 묶음을 사용할 수 있다.
.box1 .name, .box2 .name{}

우선 순위, 중복 사용

같은 요소를 여러번 선택하여 스타일을 선언하면 나중에 선언된 스타일을 우선시한다.
후손 선택자이면 더 많은 단계로 표현된 선택자가 우선시된다.
id 선택자는 class 선택자나 다른 요소 선택자보다 우선순위가 높다.
선택자가 겹쳐도 스타일이 겹쳐지지 않으면 스타일은 유지된다.

서체 종류와 스타일

스타일의 속성들

color 속성

글자 색을 지정하는 속성
body{color: #ffffff;}

font-family

폰트를 지정하는 속성
body{font-family: "돋움";}
유저가 폰트를 갖고 있지 않을 때 대안 서체를 활용.
body{font-family: "돋움", "", "";} 등등

폰트 종류

serif - 글씨에 꺽쇠가 붙어져 있다.
sans-serif - 꺽쇠가 없는 폰트
웹 폰트??

font-size

텍스트의 크기를 나타내는 속성
보통 px 단위를 사용하지만 가변적인 스타일을 위해 em, % 를 사용하기도 한다.

font-style

텍스트의 스타일을 주는 속성. 사실상 이탤릭체를 표현하기 위함을 뿐이다.
p{font-style: italic;}

font-weight

글씨의 굵기 조절하는 속성.
bold 같은 이름도 도고, 숫자도 가능
값이 inherit 이면 상위요소의 값을 상속 받는다.
p{font-weight: bold;}
p{font-weight: 300;}

line-height

줄 간격을 조절하는 속성.
글자가 차지하는 세로 길이를 의미.
px 이나 % em 사용 가능
p{line-height: 1.2;} - 1.2 em 이 줄 간격임.

font

앞의 font- 가 붙은 모든 속성을 font 속성으로 대체할 수 있다.(line-height 도 포함)
p{font: italic bolld 12px/1.5 arial, helvetica, sans-serif;}
반드시 마지막에는 font-family 속성은 넣어준다.

문자 스타일

문자와 문단에 스타일을 주는 속성

text-decoration

밑줄, 윗줄, 취소선 등이 가능
.cls{text-decoration: underline;}

text-align

문잔의 정렬을 조절하는 속성. 가운데, 좌측, 우측 정렬, 양쪽 정렬 등
p{text-align: center;}

text-indent

문단의 들여쓰기 조절.
값은 px, em 등의 수치로 넣을 수 있다.
+ 면 들여쓰기, - 면 내어쓰기.
p{text-indent: 1em;}

text-transform

영문을 모두 대문자나 소문자로 바꾸거나 capitalize 한다.
p{text-transfrom: capitalize;}

letter-spacing

글자 간 간격을 주는 속성.
px, em 단위 가능.
0 을 기준으로 양수면 멀어지고 음수면 가까워진다.
p{letter-spacing: 1em;}

word-spacing

단어 간 띄어쓰기.
letter-spacing 과 유사.
p{letter-spacing: 2em;}

vertical-align

수직정렬. 첨자 등을 의미한다.
인라인 요소에 적용한다.
윗첨자, 아래첨자, 등 위아래 위치를 조정한다.
px, % 단위로도 값을 주는것이 가능하다.
p{vertical-align: bottom;}

white-space

줄바꿈, 공백에 대한 설정.
pre 요소 처럼 보이도록 할 수 있다.
p{white-space: nowrap;}

배경 속성

요소에 배경을 넣는 속성.
웹사이트를 최적화하는데도 필요한 속성이다.
인라인, 블록 모두 가능하다.

background-color

배경에 색상을 넣는 속성.
div{background-color: #ffffff;}

background-image

배경의 이미지를 넣는 속성.
임지의 URL 주소가 들어간다. 절대 주소, 상대 주소 모두 가능하다.
div{background-image: url('');}

background-repeat

이미지를 반복시킬 것인지 아닌지.
가로, 세로 만 반복할 수 있음.
div{background-repeat: no-reap;}

background-position

배경 이미지로 넣은 이미지의 좌표 값을 정하는 속성.
px, % 등의 속성이 들어온다.
좌상단이 0,0
div{background-position: 100% 16px;} - x 100%, y 16px

background-attachment

배경이미지 고정 설정
스크롤에 따라오는지 고정되는 지 등을 정한다.
div{background-attachment: local;} - 스크롤을 따라 움직인다.

background

font 처럼 배경이미지 관련 속성들을 묶어서 표현할 수 있다.
div{bacground: url() no-repeat scoll right 50% #eee;}
블록 요소, 인라인 요소에 각각 넣어줄 수 있다.

테두리 스타일

요소에 테두리를 넣는 스타일

요소의 4 방향

모든 요소는 보통 4 방향
top bottom left right.
border-top, border-bottom 등의 이름으로 사용할 수 있다.
border-방향-요소 규칙으로 사용한다.

테두리를 구성하는 3 요소

선 굵기 - px, em 등을 사용하며 px 을 가장 많이 사용함.
p{border-top-width: 3px;}
선 형태 - 실선이나 점선 등으로 바꿔주며 없앨 수도 있다.
div{border-top-style: dotted;}
선 색상 - 선의 색상
div{border-bottom-color: #aaa;}

테두리 속성의 조합

각 테두리마다 위의 요소를 다 지정하면 너무 내용이 많이진다.
역시 속성들을 묶을 수 있다.
div{border-top: 3px dotted red;} - 상단 테두리를 설정해줌.
뿐만 아니라 형태 요소를 줄일수도 있다.
div{border-width: 3px;} - 모든 테두리 3 px
만약 두께를 다르게 하고싶다면 다음과 같이 하면 된다.
div{border-width: 3px 2px 3px 2px;} - top right bottom left 순서로 시계방향.
div{border-width: 3px 2px 2px;} - 3 개면 top, side, bottom 이 된다.
2 개면 상하, 좌우 2 개로 묶는다.
색상도 마찬가지이다.
이러한 방향 조합은 다른 속성에서도 많이 사용되므로 숙지하도록 한다.

border

다른 속성과 마찬가지로 묶음 표현이다. 가장 짧게 줄인것이다.
단 4 방향 모두 동일하게 적용된다.
div{border: 1px solid red;}

박스 모델

레이아웃을 잡을 때 기초가 되는 속성들이다.
인라인보다 블록 요소에 해당하며 높이, 너비, 여백 등이 해당한다.

margin

요소와 요소 바깥, 즉 바깥 여백을 주는 속성.
px, em, % 단위 사용
border 속성과 마찬가지로 방향에 따라 속성을 줄 수 있음.
줄여쓰기도 가능.
div{margin-top: 10px;}
div{margin: 10px 0 0 30px;}
요소 간 margin 이 겹칠 때는 큰 margin 만큼 여백이 형성된다.

padding

요소 내부에 여백을 주는 속성이다.
margin 과 동일하게 작성한다.
div{padding: 10px 0;}

width, height

블럭 요소의 너비와 넓이를 지정하는 속성.
px, em, % 단위를 사용하는데, % 는 부모 요소에 영향을 받는다.
주의할 것은 padding 이 있다면 실제로 더 커지게 된다.
실제 레이아웃 = padding + width + border

박스 모델

border, padding 을 고려해서 width 를 설정해야 한다.
이는 css3 를 활용하거나 해야한다.
