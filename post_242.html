이것이 우분투 리눅스다

리눅스

리눅스는 리누스 토르발즈가 만들었다.
정확히는 커널만 만들었다.
지금도 커널을 개발하며 업그레이드 판을 공개한다.
현재 4 버전 이상까지 나옴

배포판

공개된 커널을 쉘, 기타 유틸 등 붙여서 os 로 배포한 것.

데비안, 우분투

데비안을 변형한것이 우분투.

터미널 프롬프트 권한

$ 프롬프트 - 일반 사용자
# 프롬프트 - 루트 사용자
@ 앞에 이름이 계정.

명령어

종료

poweroff
shutdown
shutdown -P now 즉시 종료
shutdown -h 5 - 5 분 후에 꺼짐
halt -p
init 0

재부팅

shutdown -r now
reboot
init 6

로그아웃

리눅스는 멀티유저
관리자와 일반 유저가 사용한다.
logout
exit

관리자 권한 획득

sudo su
su

멀티 유저

ctrl alt f1~7 - 가상 모니터 켠다.

런 레벨

시스템의 실행 모드를 의미
init 0 - 종료
1 - 복구 모드
3 - 멀티 유저 모드
5 - xwindow 를 이용한 그래픽 모드
2,4 - 3 이랑 같다고 보면 됨.
6 - 재부팅

기본 런레벨 변경

systemd 의 default.target 을 변경하면 부팅시 변경된 런레벨 모드로 부팅된다.

자동 완성

디렉토리나 파일이 있으면 tab 눌렀을 때 자동완성 해줌
prefix 일치하는 것이 여러개 있을 때 tab 을 하나 더 누르면 모든 후보를 출력함.
도스 키 - 상하 화살표로 과거 명령어를 다시 나타낸다.

과거 명령어 출력
