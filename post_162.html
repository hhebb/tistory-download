기본 자료구조

해쉬

  • 개념
    해쉬 자체는 임의의 데이터를 고정된 길이의 데이터로 매핑해주는 것이다.
    해쉬 테이블은 해쉬값을 주소로 어떤 값을 저장하는 자료구조이다.
    key - value 쌍으로 저장이 된다.

  • 비고
    고속으로 탐색이 가능하다. $O(1)$ 의 시간복잡도를 가진다.
    보통 입력 데이터를 key 값으로 만들어준 후 해쉬 함수에 넣어서 해쉬 주소(값) 을 만든다.
    이 해쉬 주소에 해당하는 곳에 담고자 하는 값 value 를 저장해 놓으면 다음 번에 key 에 해당하는 데이터를 이용해 빠른 속도로 접근할 수 있는 것이다.
    해쉬 함수에는 여러가지가 쓰일 수 있다.
    충돌이 안일어나도록 하는 것이 중요하다.

    일반 선형 데이터를 해쉬로 바꾸면 탐색에 매우 빠를 수 있다!

    참고로 해쉬와 해쉬 테이블은 다르다.
    해쉬 값은 해쉬 함수를 통과한 값이고, 해쉬 테이블은 그 해쉬 값이 인덱스에 해당하는 하나의 저장소이다.
    혼용해서 쓰이기도 하는 듯 하다.

# data --> key 값
def make_key(data):
    key = ord(data[0])
    return key

# key 값 --> 해쉬 주소
def make_hash(key):
    hash_value = key % 10
    return hash_value

# 해당 해쉬 주소에 값 저장
def save(data, value):
    key = make_key(data)
    hash_address = make_hash(key)
    hash_table[hash_address] = value

# 해당 데이터에 있는 값을 확인
def find(data):
    key = make_key(data)
    hash_address = make_hash(key)
    return hash_table[hash_address]

hash_table = [0] * 10

data = 'surf'
save('minsu', 100)
save('jisu', 40)
