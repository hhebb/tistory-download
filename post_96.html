기초 정렬.

수 정렬하기

  • 난이도 - 하
  • 유형 - 하
  • 비고
    n = 1000 이라서 sort() 를 써도 되고 기본 정렬 알고리즘 써도 됨.
    파이썬 sort 는 nlogn 보장함.
n = int(input())
lst = []

for _ in range(n):
    lst.append(int(input()))

def bubble_sort(arr):
    for i in range(len(arr)-1, 0, -1):
        for j in range(i):
            if arr[j+1] < arr[j]:
                arr[j+1], arr[j] = arr[j], arr[j+1]
    return arr
bubble_sort(lst)

소트인사이드

  • 난이도 - 하
  • 유형 - 정렬, 배열
  • 비고
    그냥 쉬움.
    강사는 시시한 방법이 아닌 조금 재미있게 문제를 풀었다.
    뒤에서부터 큰 수를 찾아 하나씩 출력하게했다.
# 소트 인사이드
n = input()
splited = [i for i in n]
splited.sort(reverse=True)
print(''.join(splited))
