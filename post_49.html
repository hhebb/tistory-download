출처

프로그래머스

난이도

프로그래머스 level 2

유형

문자열

특이사항

공백을 처리해 줘야함.

총평

매우 간단한 문제인데 문제 설명이 부실하다고 생각함.
공백문자에 대해 의심이 갔지만 문제에 명시되어 있지 않아 알고리즘의 문제라고 인식하고 계속 헛발질 함.

def solution(s):
    answer = ''
    char_list = []
    for char in s:
        char_list.append(char)

    word_started = False

    for idx, char in enumerate(char_list):
        if char != ' ':
            if not word_started:
                word_started = True
                char_list[idx] = char.upper()
            else:
                char_list[idx] = char.lower()
        else:
            word_started = False

    answer = ''.join(char_list)
    return answer
