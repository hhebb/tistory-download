MVC

출처 - 얄팍한 코딩사전
백엔드 프레임워크라고 보면된다.
즉, 서버 개발 전반을 담당한다.

Model

  • 데이터에 관련된 것이라고 보면 됨.
  • 데이터의 형식 지정, 저장 및 불러오기 기능을 담당.

View

  • Model 의 데이터를 나타나게 하는 방식 등을 담당
  • html, css 등의 요소가 여기에 작성됨.

Controller

  • Model, View 를 연결해주는 역할
  • Model 데이터를 View 에 연결하여 사용자가 조작할 수 있도록 전반적 제어.

MVC 프레임워크

  • 웹 서비스에 필요한 위의 기능들을 모두 모아놓은 프레임워크를 제공한다.
  • 종류
    spring
    express
    laravel
    django
    rails
    dotnet core
    play 등

역할

  • 백엔드 프레임워크들은 다 비슷한 역할과 기능이 있다.
  • 이들은 동적웹을 개발하는데 필요한 기능들을 제공한다.
  • 코드 내부에 html 과 각자 언어를 섞어서(가공/요리) 최종적으로 브라우저에 보이게 하는 것을 포함한다.
  • 라우팅도 포함한다. 프레임워크마다 방식이 다르다.
