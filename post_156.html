새 강의

명령어 관련

powershell 은 리눅스와 명령어가 거의 같다.
npm init - 설정 파일을 만들어 패키지를 만든다. package.json 파일 만들어서 저장한다.
npm install name --save - 패키지 파일 설치하고 설정파일에 기록한다.
npm run cmd - 실행
package 파일에 'scripts' 에 명령어를 등록하고 실행할 파일을 적어두면 npm run _command_하면 돌아간다.

nodemon

소스를 변경할 때 그걸 감지해서 자동으로 서버를 재시작해주는 툴
