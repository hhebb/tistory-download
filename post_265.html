파이썬 모듈

  • 모듈은 함수나 변수, 클래스를 모아 놓은 파일
  • 현재 디렉터리에 있는 모듈이나 라이브러리 패스에 있는 모듈만 불러올 수 있다.
  • __name__ 변수 - 내부용 변수. 직접 실행 시 이 값은 __main__ 이 된다.
    • 다른 파일에서 모듈로 임포트 할 땐 파일이름으로 들어감.
  • PATH 설정 필요함. - PYTHONPATH 에 추가한다.

패키지 구성

  • 루트 디렉터리
  • __init__ 파일 - 해당 디렉터리가 패키지 일부임을 알려줌
    • 파이썬 3.3 부터는 init 파일 없어도 인식한다.
    • 하지만 호환성을 위해 생성하도록 한다.

패키지

  • import 는 모듈 혹은 패키지만 가능하다.
